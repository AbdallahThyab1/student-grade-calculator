## 📌 Overview

A simple web application that allows users to:

- Enter multiple subject marks
- Dynamically add or remove inputs
- Calculate the average instantly
- 🎯 Practice DOM manipulation and interactive UI
- 🚀 Built using pure JavaScript (no frameworks)
